NotifyMe

A personal Android receiver for Firebase alerts with saved history, a device API key for scripted sends, and website-hosted updates outside the Play Store.

How To Use API Download APK
Android 8.0+ Kotlin + Compose FCM + Website Updates

Reliable visible delivery, no raw FCM token sharing

NotifyMe now exposes a managed API Keys screen instead of asking you to share the raw token. Fren stores the Firebase delivery credential, sends normal high-priority notifications for better lock-screen reliability, syncs interactive actions into app history, and broadcasts new app updates to registered devices.

What It Does

Receive Firebase Cloud Messaging alerts, save them to local history, trigger update installs from the web, route notifications through your own backend, and let users send confirmations or replies from the app after a notification is received.

Core Stack

  • Kotlin
  • Jetpack Compose Material 3
  • Firebase Cloud Messaging
  • Website-hosted APK updates

Best For

Personal push testing, webhook-driven alert delivery, and lightweight Android installs that can be shared through a direct backend link.

Daily Usage

Successful NotifyMe sends per day, streamed from the fren backend.
