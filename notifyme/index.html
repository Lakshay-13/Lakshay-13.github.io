NotifyMe

A clean personal Android app for receiving Firebase push notifications, checking tokens, keeping local notification history, and updating directly from the web.

Download APK 1.0.2 View Live Manifest
Android 8.0+ Kotlin + Compose FCM + OTA Updates

Built for direct notification testing and delivery control

NotifyMe surfaces the device token, stores received alerts, supports local test sends, and updates from the same website backend that serves the APK.

What It Does

Receive Firebase Cloud Messaging alerts, inspect the current token, view notification history inside the app, and manage updates without going through the Play Store.

Core Stack

  • Kotlin
  • Jetpack Compose Material 3
  • Firebase Cloud Messaging
  • Website-hosted APK updates

Best For

Personal push testing, webhook-driven alert delivery, and lightweight Android installs that can be shared directly through a link.
